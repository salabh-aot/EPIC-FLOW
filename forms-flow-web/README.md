# Building using Docker-Compose
```cd /forms-flow-web
    docker-compose build --build-arg FORMFLOW_SOURCE_REPO_URL=https://github.com/aot-technologies/forms-flow-ai --build-arg FORMFLOW_SOURCE_REPO_BRANCH=develop forms-flow-web
```
